## How to Use These Examples

1. **Copy** the entire `places` array from any example above
2. **Open** `assets/data/places.json`
3. **Replace** the entire content with the example (or merge the places array)
4. **Save** the file
5. **Hot reload** the app (Cmd+S or Ctrl+S)

## Customization Tips

- **Change coordinates** to your actual location
- **Adjust distances** to match real measurements
- **Update emojis** to better match your venue
- **Modify descriptions** to provide helpful information
- **Update section names** to match your location layout

## Tips for Real Deployments

- Use real GPS coordinates from Google Maps
- Calculate distances using the Haversine formula
- Add authentication to load dynamic place lists
- Connect to a real backend API instead of static JSON
- Implement real AR using ARCore/ARKit
